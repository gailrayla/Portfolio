# gailparayno.com

Personal portfolio of Gail Parayno — software engineer, frontend focused, with a design degree.

Built with Next.js (App Router), TypeScript, Tailwind, and GSAP. Smooth scroll by Lenis; type set in Bricolage Grotesque and Space Grotesk.

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
